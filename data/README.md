# Data Layer

## Overview
This folder contains datasets used for workload and resource utilization analysis.

## Data Sources

### Planned Allocation
- Planned working hours
- Employee project assignments

### Actual Timesheets
- Actual worked hours
- Recorded execution data

### Referenced Data
- Employee Register
- Projects

## Data Structure

Key fields:
- employee_id
- project_id
- date
- planned_hours
- actual_hours
- capacity

## Notes

Data is generated to simulate real resource management scenarios.
